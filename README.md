# thien-nhan — HR assessment fusing Western psychology and Eastern physiognomy

> DISC + Big Five + MBTI cross-validated with Eastern and Western face-reading frameworks. Covers the full recruitment funnel from personality screening to structured interview.

[![OpenClaw Skill](https://img.shields.io/badge/OpenClaw-Skill-blueviolet)](https://github.com/NachaFromMars)

## Overview
thien-nhan is a comprehensive HR assessment system combining validated Western psychometrics with Eastern and Western physiognomy. Runs DISC, Big Five OCEAN, and MBTI with cross-validation, then optionally extends into face reading and AI vision analysis. STAR-based behavioral interview analysis completes the toolkit.

## Features
**Psychometrics:** DISC (40Q, 4 profiles, 12 combined, 8 role-fit) · Big Five OCEAN (50Q, 5 factors) · MBTI (40Q, 16 types) · Combined Profile cross-validation

**Physiognomy — Eastern:** Ngũ Quan, Tam Đình, Tứ Khí Chất, Ngũ Nhạc, Thập Nhị Cung
**Physiognomy — Western:** Four Temperaments, Face Shapes, Ekman Micro-Expressions, Eye Analysis
**AI Vision:** structured prompts for face reading · DeepFace bridge (optional, age/gender/emotion)

**Interview:** STAR behavioral analysis
